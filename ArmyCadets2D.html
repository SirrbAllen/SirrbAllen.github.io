Army Cadets 2D Game

Project Details

This project was completed for the Army Cadet Force. The idea was that if the army cadets could show potential recruits a fun game, they might be more willing to join the ACF.

My main roll in this project was designing the sprites. The game is a 2D 8-bit style abseiling game that sees the player descend down a cliff face at increasing speeds whilst avoiding obstacles.

The Game

This game was built using JavaScript. The Phaser game library was used to implement physics and game mechanics.

The sprites were all made in Photoshop. I created these in an 8-bit pixelated style that looks fun and interesting.
