Theo's Trail

Project Details

This was a project for the Sheffield Children's Hospital. The aim was to create a game that would encourage people to be active whilst raising awareness for their Golden Hour Campaign to raise money for a helipad.

The aim of the game is to walk around the Olympic Legacy Park in Sheffield to different points of interest. Then, in augmented reality, the hospital's mascot appears on screen doing various exercises.

The Game

The game was built using Unity game engine and used C# as the main language. My main roll was to create the onscreen assets for the game and to help with some of the backend processing.

We worked with some fairly large companies and charities for this project such as MoveMore, AMRC, and the Olympic Legacy Park.
